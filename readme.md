The Suggestions Bot
---

This bot represents the third version of the code base for [suggestions.gg](https://suggestions.gg). This project has been retired and the platform now runs on the following repository: [https://github.com/suggestionsbot/suggestions-v4](https://github.com/suggestionsbot/suggestions-v4)

While it is open source we do not provide support for self-hosting. Self hosting this code is likely easier then version four.
